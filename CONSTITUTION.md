# Constitution

1. This project is a static website.
2. The codebase must use plain CSS and plain JavaScript only.
3. No bundling, transpilation, or compilation is allowed.
4. Source files must be served and run directly as written.
5. CSS must use modern CSS features and standards.
6. JavaScript must use modern JavaScript (ES modules and current language features).
7. Any change that violates these rules must be rejected.
